# Expense Tracker Dashboard

## 📌 Project Overview

Expense Tracker Dashboard is a web application developed using PHP and MySQL that helps users manage their personal finances. Users can securely register, log in, add income and expense transactions, edit or delete records, search transactions, and view financial summaries with interactive charts.

---

## 🚀 Features

- User Registration & Login Authentication
- Secure Logout
- Add Income & Expense Transactions
- Edit Existing Transactions
- Delete Transactions
- Transaction History
- Search Transactions
- Filter by Transaction Type
- Dashboard with Financial Summary
- Total Income
- Total Expense
- Current Balance
- Total Number of Transactions
- Expense Category Pie Chart
- Income vs Expense Bar Chart
- Responsive and Modern User Interface
